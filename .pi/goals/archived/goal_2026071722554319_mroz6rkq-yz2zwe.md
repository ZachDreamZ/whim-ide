OBJECTIVE

The final experience should feel like a professional desktop coding agent:

New task
→ opens a clean agent conversation
→ user sends a natural-language goal
→ agent executes inside the conversation

→ environment and source information appear in a collapsible right sidebar

Do not send the user to a task dashboard.

Do not expose raw orchestration data as the main interface.

Do not create repeated conversations named “continue.”
